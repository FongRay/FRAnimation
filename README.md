> IconFont 是使用字体文件替代图片来保存图标的技术，可用于Web、Android和iOS。
* [Icon Font](http://www.iconfont.cn)
* [使用IconFont减小iOS应用体积](http://johnwong.github.io/mobile/2015/04/03/using-icon-font-in-ios.html)

用在移动端时，具有以下特点:
#### 优点
* 减小体积，字体文件比图片要小
* 图标保真缩放，解决2x/3x乃至将来的nx图问题
* 方便更改颜色大小，图片复用
#### 缺点
* 只适用于纯色icon
* 使用unicode字符难以理解
* 需要维护字体库
